Projects

Mocial: A Movie Review App

GitHub Repository

Mocial is an Android app which helps a user to review a movie and find the users with same interests in a particular genre. This app comprises of Android, Python Script, Firebase, NodeJS and MongoDB. The app contains the user interface which takes the data from the user and sends it to Firebase which is connected to the Python Script. An event listener is applied on the Firebase node so that Python script can easily retrieve the data and process it and give back the result that will be a “Positive Review” or a “Negative Review”.
